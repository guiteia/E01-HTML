O que são estruturas condicionais?

Chamamos de estruturas condicionais as instruções para testar se uma condição é verdadeira ou não.

Entre as principais estruturas condicionais em C temos:
  • If
  • If-else
  • Switch

A estrutura if

A estrutura if é uma das principais, usamos ela da seguinte forma: if (condição) {instrução;}

Exemplo

if (num==1) {
printf("Você escolheu a opção 1");
}

Nesse caso, SE o usuário digitar o número 1, a mensagem impressa será "Você escolheu a opção 1"

A estrutura if-else

Com a estrutura if-else, caso a entrada não satisfaça a condição inicial, podemos usar o else como uma alternativa.

Exemplo

if (num==1) {
printf("Você escolheu a opção 1");
}
else {
printf("Você não escolheu a opção 1");
}

Nesse caso, SE o usuário digitar o número 1, a mensagem impressa será "Você escolheu a opção 1", mas se digitar outro valor a mensagem a ser imprimida será "Você não escolheu a opção 1".

A estrutura switch

Com a estrutura switch podemos analisar vários valores e com base nestes dizer qual será a instrução.

Exemplo

switch (num)
{
case 1:
printf("Você escolheu a opção 1");
break;

case 2:
printf("Você escolheu a opção 2");
break;

case 3:
printf("Você escolheu a opção 3");
break;
}
Nesse caso, SE o usuário digitar o número 1, a mensagem impressa será "Você escolheu a opção 1" e se digitar o número 2 receberá a mensagem "Você escolheu a opção 2", o mesmo acontece se o usuário digitar 3. O switch é útil quando podemos aceitar diferentes valores.
